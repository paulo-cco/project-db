# Exercícios de SQL

Este repositório reúne atividades práticas desenvolvidas com o objetivo de aprimorar conhecimentos em SQL e manipulação de bancos de dados relacionais.

## Objetivo

Praticar os principais conceitos de SQL por meio da criação e execução de consultas e operações em banco de dados, desenvolvendo habilidades essenciais para aplicações que utilizam persistência de dados.

## Conteúdos Praticados

- Criação de tabelas (`CREATE TABLE`)
- Inserção de registros (`INSERT`)
- Consulta de dados (`SELECT`)
- Atualização de registros (`UPDATE`)
- Remoção de registros (`DELETE`)
- Definição de chaves primárias e estrangeiras
- Relacionamentos entre tabelas
- Filtros com `WHERE`
- Ordenação de resultados com `ORDER BY`
- Funções de agregação (`COUNT`, `SUM`, `AVG`, `MAX`, `MIN`)
- Junções entre tabelas (`JOIN`)

## Operações CRUD

As atividades contemplam a prática das quatro operações fundamentais de bancos de dados:

| Operação | Comando SQL |
|-----------|------------|
| Create | INSERT |
| Read | SELECT |
| Update | UPDATE |
| Delete | DELETE |

## Estrutura

Cada atividade aborda diferentes cenários de banco de dados, permitindo exercitar consultas, manipulação de registros e modelagem básica de dados.

## Tecnologias Utilizadas

- SQL
- MySQL
